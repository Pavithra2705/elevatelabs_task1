# Task 1: Understanding Dataset & Data Types  
## Dataset: Titanic

---

## Objective
To understand the structure, data types, and machine learning readiness of the Titanic dataset using Python.

---

## Tools Used
- Python  
- Pandas  
- NumPy  
- Jupyter Notebook / Google Colab  

---

## Dataset Overview
- Rows: 891  
- Columns: 12  
- Target Variable: `Survived`

---

## Key Steps
- Loaded and inspected dataset using Pandas  
- Identified numerical, categorical, ordinal, and binary features  
- Analyzed missing values and data imbalance  
- Checked dataset suitability for machine learning  

---

## Observations
- Missing values present in Age, Cabin, and Embarked  
- Target variable shows slight imbalance  
- Data requires preprocessing before modeling  

---

## Outcome
Gained clear understanding of data structure, feature types, and ML readiness.
